Privacy Policy

1. Introduction

utsushi (hereinafter "this tool") is a tool for comparing text differences in your browser. This privacy policy explains how information is handled in this tool.

2. Handling of Text Data

Text data entered into this tool is processed entirely locally within your browser.

  • ‚úÖ Input content is not sent to any server
  • ‚úÖ Input content is not saved or recorded
  • ‚úÖ Input data is deleted when you close the page

3. About Analytics

This tool uses the following analytics tools for service improvement.

  • Google Analytics: Collects statistical information such as visitor count, page views, and device usage
  • Microsoft Clarity: Collects anonymized user interaction patterns (clicks, scrolls, etc.)

These tools may use cookies, but the content of your text input is never collected.

Cookies may also be used to create advanced monitoring reports based on tracked events (such as user timelines or funnel reports).

4. About Cookies

This tool uses cookies or local storage for the following purposes.

  • Saving theme settings (dark/light mode)
  • Saving language settings
  • Analytics (see above)

5. Third-Party Disclosure

Since this tool does not collect input data, no disclosure to third parties is made. Statistical information collected through analytics is handled according to each analytics service's privacy policy.
